## 🎯 这是什么？

**TRAE Learning Projects**（学习园区）是一个专注于 **Vibecoding**（AI 辅助编程）进阶学习的开源文档项目。我们提供系统化的学习路径、优质的学习资源和实战教程，帮助开发者掌握 AI 辅助编程的核心技能。

### 💡 为什么重要

我们提供系统化的学习资源，帮助开发者：

- 🔍 **了解** Vibecoding 的概念与可能性
- 📚 **学习** 真实的项目实现
- 💡 **获得** 开发思路和技术方案的灵感
- 🌟 **展示** 你的作品并获得社区反馈

## 🚀 快速开始

### 本地预览

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run docs:dev

# 构建生产版本
npm run docs:build
```

访问 [TRAE Learning Docs](https://trae-community.github.io/trae-learning-projects/) 查看在线文档。

## 📚 学习内容

### 什么是 Vibecoding？

Vibecoding 是一种新兴的编程范式，开发者通过与 AI 协作，以自然语言描述需求，AI 生成代码，开发者专注于审查和优化。这种模式大大提升了开发效率，让开发者能够更专注于创造性工作。

- [📖 文档指南](./guide/) - 核心概念与理念
